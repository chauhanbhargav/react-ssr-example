# React SSR Example using `create-react-app`

This is the simple example of React SSR with node js server.

## Libraries used
- Create React App
- React (16.0.0 - "Fiber")
- Redux (3.7.2)
- Thunk (2.2.0)
- React Helmet (5.2.0)
- React Router v4
- React Router Redux (5.0.0-alpha.8)
- Ant Design

## Steps to run the project
- Clone the Project
- `cd react-ssr` 
- `npm install`
- `npm run build`
- `npm run serve`
